Featured Projects

jChecker-engine

jChecker는 Java Programming Language를 사용해, 객체지향 프로그램 자동 채점을 진행하는 도구입니다.

현재 한동대학교 자바 프로그래밍 언어 수업에서 사용되고 있습니다.

https://github.com/HandongSF/jChecker-Engine

WICWIU

WICWIU는 국내 대학 최초로 공개하는 딥러닝 오픈소스 프레임워크입니다.

WICWIU는 모든 API가 C++로 제공되어 메모리 및 성능 최적화에 유리합니다. 또한, 응용시스템의 개발 외에도 프레임워크 자체를 특수한 환경에 맞도록 수정 및 확장이 가능합니다.

https://github.com/HandongSF/WICWIU

histudy-fe

스터디 그룹 웹앱 프론트엔드 프로젝트입니다.

스터디 그룹 매칭과 보고서 관리 등 효과적인 UX/UI를 제공하는 것에 초점을 맞추고 있습니다.

https://github.com/HandongSF/histudy-fe

histudy-be

스터디 그룹 웹앱 백엔드 프로젝트입니다.

스터디 진행에 관련한 모든 백엔드 작업을 처리하는 기능을 수행합니다.

https://github.com/HandongSF/histudy-be

Incubating Projects

ByteTok

JVM의 바이트코드를 기준으로 Large Language Model을 학습하기 위한 Tokenizer입니다.

자연어와 바이트코드의 차이점을 이용해 새로운 Tokenizer 기법을 연구합니다.

https://github.com/HandongSF/ByteTok

EnCus

EnCus는 Automated Program Repair (APR)에서 패치를 생성하기 위한 검색 공간을 맞춤화하는 새로운 접근 방식입니다.

여러 오픈 소스 프로젝트 풀과 AST 수준의 코드 차이 도구로부터 수정 연산을 모아 사용하는 앙상블 방식의 접근을 활용하여 검색 공간을 줄입니다.

KoreanUnificationParallelCorpus

북한과 남한은 같은 한국어를 사용하지만 분단으로 인해 그 형태가 많이 달라졌습니다.

해당 차이를 극복하기 위해서 여러 단어와 문장에 대한 데이터셋을 확보하여 추후 연구에 사용될 수 있도록 합니다.

https://github.com/HandongSF/KoreanUnificationParallelCorpus

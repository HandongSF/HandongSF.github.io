github에 관하여

이 페이지는 HSF에 개발자들을 위한 github 자료를 제공합니다.

깃허브에 대한 정확한 정보를 통해 보다 오픈소스 개발 중 발생할 수 있는 일들을 방지하는 것을 목표로 둡니다.

깃허브가 무엇인가요?

깃허브(GitHub)는 소프트웨어 개발자들이 소스 코드를 공유하고 협업하기 위한 플랫폼입니다.

깃(Git)이라는 버전 관리 시스템을 사용하여 프로젝트의 여러 버전을 관리할 수 있게 해주며, 개발자들이 코드를 저장하고, 추적하며, 변경 사항을 다른 사용자들과 공유할 수 있게 해줍니다.

깃허브는 오픈 소스 프로젝트부터 개인 프로젝트까지 다양한 종류의 소프트웨어 개발 프로젝트를 지원합니다.
